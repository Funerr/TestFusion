---
name: disable-airplane-mode
description: Use when 需要在鸿蒙 4.2 设备上关闭飞行模式，并确保后续网络能力可恢复。
call_mcp: true
parameters:
  method:
    type: string
    required: false
    description: 执行方式，可选 auto、control_center 或 settings，默认 auto
  verify:
    type: boolean
    required: false
    description: 是否在执行后校验飞行模式状态，默认 true
---

# Implementation

1. 打开控制中心或系统设置中的网络页。
2. 执行 `mobile_toggle_system_setting`，传入 `setting="airplane_mode"` 和 `enabled=false`。
3. 通过 `mobile_assert_system_state` 校验飞行模式已关闭。

# Examples

## Example 1 - 关闭飞行模式
**Input:**
```json
{"method":"auto","verify":true}
```
