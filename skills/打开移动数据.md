---
name: enable-mobile-data
description: Use when 需要在鸿蒙 4.2 设备上打开移动数据，并把抽象语义落地为控制中心或设置页操作。
call_mcp: true
parameters:
  method:
    type: string
    required: false
    description: 执行方式，可选 auto、control_center 或 settings，默认 auto
  verify:
    type: boolean
    required: false
    description: 是否在执行后校验移动数据状态，默认 true
---

# Implementation

1. 优先打开控制中心，必要时进入移动网络设置页。
2. 执行 `mobile_toggle_system_setting`，传入 `setting="mobile_data"` 和 `enabled=true`。
3. 通过 `mobile_assert_system_state` 校验移动数据已开启。

# Examples

## Example 1 - 打开移动数据
**Input:**
```json
{"method":"control_center","verify":true}
```
