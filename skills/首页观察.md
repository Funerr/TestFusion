---
name: observe-home
description: Use when 需要在首页定位搜索框或关键入口，为点击、观察或首页断言提供更稳定的定位语义。
call_mcp: true
parameters:
  action:
    type: string
    required: false
    description: 目标动作，可选 observe 或 click，默认 observe
  target:
    type: string
    required: false
    description: 首页目标元素标识，默认 search_box
  verify:
    type: boolean
    required: false
    description: 是否在执行后补充页面或元素校验，默认 false
---

# Implementation

1. 优先使用首页已知的 resource-id 或文本定位关键控件。
2. 当 `action="click"` 时，执行 `mobile_click_by_id` 或 `mobile_click_by_text`。
3. 当 `action="observe"` 时，结合截图、前台页面和元素信息输出首页观察结果。

# Examples

## Example 1 - 观察首页搜索框
**Input:**
```json
{"action":"observe","target":"search_box","verify":false}
```
