---
name: open-system-settings-page
description: Use when 需要在鸿蒙 4.2 设备上进入 WLAN、蓝牙、显示、声音等系统设置二级页面。
call_mcp: true
parameters:
  page:
    type: string
    required: true
    description: 目标系统页，可选 wlan、bluetooth、display、sound、location、nfc 等
  verify:
    type: boolean
    required: false
    description: 是否在执行后校验当前页面，默认 true
---

# Implementation

1. 如未在设置首页，先打开系统设置。
2. 执行 `mobile_open_system_page` 并传入 `page`。
3. 通过页面标题、关键开关或 `current_screen` 确认进入目标二级页。

# Examples

## Example 1 - 打开蓝牙设置页
**Input:**
```json
{"page":"bluetooth","verify":true}
```
