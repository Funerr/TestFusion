---
name: adjust-volume
description: Use when 需要在鸿蒙 4.2 设备上调整媒体音量或铃声音量，并将抽象描述映射为统一的系统值操作。
call_mcp: true
parameters:
  stream:
    type: string
    required: false
    description: 音量类型，可选 media 或 ring，默认 media
  volume_level:
    type: integer
    required: true
    description: 目标音量百分比，范围 0 到 100
  verify:
    type: boolean
    required: false
    description: 是否在执行后校验音量值，默认 true
---

# Implementation

1. 根据 `stream` 确定目标系统值键，媒体音量对应 `media_volume`，铃声音量对应 `ring_volume`。
2. 执行 `mobile_set_system_value`，传入对应 `setting` 和目标百分比。
3. 通过 `mobile_get_system_state` 或 `mobile_assert_system_state` 校验音量值一致。

# Examples

## Example 1 - 调整媒体音量到 35%
**Input:**
```json
{"stream":"media","volume_level":35,"verify":true}
```
