---
name: open-notification-center
description: Use when 需要在鸿蒙 4.2 设备上打开通知中心，以便观察通知、清空消息或验证下拉面板状态。
call_mcp: true
parameters:
  verify_elements:
    type: boolean
    required: false
    description: 是否在打开后检查通知项或清除入口，默认 false
---

# Implementation

1. 执行 `mobile_open_notification_center`。
2. 观察通知列表、时间信息或“清空”入口是否已出现。
3. 后续通知相关断言和点击优先在该面板内完成。

# Examples

## Example 1 - 打开通知中心
**Input:**
```json
{"verify_elements":true}
```
