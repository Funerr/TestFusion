call_mcp: true
parameters:
  method:
    type: string

  verify:
    type: boolean
    required: false
    description: 是否在执行后校验蓝牙状态，默认 true
---

# Implementation

1. 优先下拉控制中心并定位蓝牙开关。
2. 如果控制中心没有入口，则打开系统设置并进入蓝牙页。
3. 执行 `mobile_toggle_system_setting`，传入 `setting="bluetooth"` 和 `enabled=true`。
4. 通过 `mobile_assert_system_state` 或系统页面开关状态确认蓝牙已开启。

# Examples

## Example 1 - 开启蓝牙
**Input:**
