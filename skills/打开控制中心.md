name: open-control-center
description: Use when 需要在鸿蒙 4.2 设备上打开控制中心，以便执行快捷开关、亮度或音量相关动作。
call_mcp: true
parameters:
  verify_elements:
    type: boolean
    required: false
    description: 是否在展开后检查快捷开关元素是否可见，默认 false
---

# Implementation

1. 执行 `mobile_open_control_center` 或兼容的 `mobile_open_quick_settings`。
2. 观察快捷开关、亮度条或音量条是否已出现。
3. 后续系统开关动作优先复用当前面板，减少重复导航。

# Examples

## Example 1 - 打开控制中心
**Input:**
```json
{"verify_elements":true}
```
