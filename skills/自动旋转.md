---
name: toggle-auto-rotate
description: Use when 需要在鸿蒙 4.2 设备上打开或关闭自动旋转，并要求统一解释控制中心与显示设置路径。
call_mcp: true
parameters:
  state:
    type: string
    required: true
    description: 目标状态，可选 on 或 off
  method:
    type: string
    required: false
    description: 执行方式，可选 auto、control_center 或 settings，默认 auto
  verify:
    type: boolean
    required: false
    description: 是否在执行后校验自动旋转状态，默认 true
---

# Implementation

1. 打开控制中心或显示设置页。
2. 执行 `mobile_toggle_system_setting`，传入 `setting="auto_rotate"` 和对应 `enabled` 值。
3. 通过 `mobile_assert_system_state` 校验自动旋转状态与 `state` 一致。

# Examples

## Example 1 - 关闭自动旋转
**Input:**
```json
{"state":"off","method":"control_center","verify":true}
```
