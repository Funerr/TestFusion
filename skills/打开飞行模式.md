---
name: enable-airplane-mode
description: Use when 需要在鸿蒙 4.2 设备上打开飞行模式，并要求执行后可校验系统网络状态切换结果。
call_mcp: true
parameters:
  method:
    type: string
    required: false
    description: 执行方式，可选 auto、control_center 或 settings，默认 auto
  verify:
    type: boolean
    required: false
    description: 是否在执行后校验飞行模式状态，默认 true
---

# Implementation

1. 打开控制中心或网络设置页。
2. 执行 `mobile_toggle_system_setting`，传入 `setting="airplane_mode"` 和 `enabled=true`。
3. 通过 `mobile_assert_system_state` 校验飞行模式已开启。

# Examples

## Example 1 - 打开飞行模式
**Input:**
```json
{"method":"auto","verify":true}
```
