---
name: adjust-brightness
description: Use when 需要在鸿蒙 4.2 设备上把亮度调到指定百分比，并通过系统值断言保证结果可验证。
call_mcp: true
parameters:
  brightness_level:
    type: integer
    required: true
    description: 目标亮度百分比，范围 0 到 100
  method:
    type: string
    required: false
    description: 执行方式，可选 control_center、settings 或 direct，默认 direct
  verify:
    type: boolean
    required: false
    description: 是否在执行后校验亮度值，默认 true
---

# Implementation

1. 根据 `method` 选择控制中心、显示设置页或直接系统值写入。
2. 执行 `mobile_set_system_value`，传入 `setting="brightness"` 和目标百分比。
3. 通过 `mobile_get_system_state` 或 `mobile_assert_system_state` 校验亮度值一致。

# Examples

## Example 1 - 调整亮度到 80%
**Input:**
```json
{"brightness_level":80,"method":"direct","verify":true}
```
