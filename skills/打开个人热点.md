---
name: enable-hotspot
description: Use when 需要在鸿蒙 4.2 设备上打开个人热点，并自动选择控制中心或设置页的稳定路径。
call_mcp: true
parameters:
  method:
    type: string
    required: false
    description: 执行方式，可选 auto、control_center 或 settings，默认 auto
  verify:
    type: boolean
    required: false
    description: 是否在执行后校验热点状态，默认 true
---

# Implementation

1. 打开控制中心或个人热点设置页。
2. 执行 `mobile_toggle_system_setting`，传入 `setting="hotspot"` 和 `enabled=true`。
3. 通过 `mobile_assert_system_state` 校验热点已开启。

# Examples

## Example 1 - 打开个人热点
**Input:**
```json
{"method":"auto","verify":true}
```
