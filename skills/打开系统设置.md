name: open-system-settings
description: Use when 需要在鸿蒙 4.2 设备上打开系统设置首页，作为进入二级设置页或执行系统配置的起点。
call_mcp: true
parameters:
  verify:
    type: boolean
    required: false
    description: 是否校验当前页面已经进入系统设置首页，默认 true
---

# Implementation

1. 执行 `mobile_open_system_settings` 打开系统设置首页。
2. 观察页面标题、搜索框或设置列表，确认已经到达正确页面。
3. 后续如需进入二级设置页，继续调用 `mobile_open_system_page`。

# Examples

## Example 1 - 打开系统设置
**Input:**
```json
{"verify":true}
```
