---
name: enable-location
description: Use when 需要在鸿蒙 4.2 设备上打开定位服务，并将自然语言步骤落地为控制中心或系统设置操作。
call_mcp: true
parameters:
  method:
    type: string
    required: false
    description: 执行方式，可选 auto、control_center 或 settings，默认 auto
  verify:
    type: boolean
    required: false
    description: 是否在执行后校验定位状态，默认 true
---

# Implementation

1. 打开控制中心或定位设置页。
2. 执行 `mobile_toggle_system_setting`，传入 `setting="location"` 和 `enabled=true`。
3. 通过 `mobile_assert_system_state` 校验定位已开启。

# Examples

## Example 1 - 打开定位
**Input:**
```json
{"method":"auto","verify":true}
```
