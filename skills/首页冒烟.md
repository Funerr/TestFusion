---
name: home-smoke
description: Use when 需要启动应用并快速确认首页可达，尤其适合“看看首页是不是正常”这类模糊烟测描述。
call_mcp: true
parameters:
  package:
    type: string
    required: false
    description: 目标应用包名，未传时沿用当前运行配置
  activity:
    type: string
    required: false
    description: 目标首页 Activity，未传时沿用当前运行配置
  verify:
    type: boolean
    required: false
    description: 是否在执行后校验首页关键元素，默认 true
---

# Implementation

1. 执行 `mobile_launch_app` 启动目标应用。
2. 观察前台应用、当前页面和首页关键元素。
3. 如果首页元素清晰可判定，则执行断言；如果原始需求仍模糊，则补充 `NEEDS_REVIEW` 提示。

# Examples

## Example 1 - 首页烟测
**Input:**
```json
{"package":"com.demo.app","activity":"MainActivity","verify":true}
```
